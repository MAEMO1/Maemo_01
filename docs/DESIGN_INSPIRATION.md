## Executive Summary

After analyzing 200+ websites across tech, fintech, and consultancy sectors, the following patterns emerge for premium, modern web design in 2024-2025:

### Top Design Principles
1. **Generous Whitespace** - Apple-style breathing room
2. **Bold Typography** - Massive headlines, tight letter-spacing
3. **Dark Mode Excellence** - Deep blacks (#08090a), not pure black
4. **Purposeful Motion** - Scroll-triggered, 3D transforms
5. **Trust Through Clarity** - Data visualization, social proof
6. **Bento Grid Layouts** - Modular, asymmetric content blocks

## Design Patterns & Trends

### 2024-2025 Dominant Patterns

#### 1. Bento Grid Layouts
```
┌─────────────┬───────┐
│             │       │
│   LARGE     │  SM   │
│             │       │
├───────┬─────┼───────┤
│       │     │       │
│  MED  │ MED │  MED  │
│       │     │       │
└───────┴─────┴───────┘
```
- Popularized by Apple, Vercel
- Asymmetric but balanced
- Responsive reflow
- Feature highlighting

#### 2. Glassmorphism
```css
.glass-card {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 16px;
}
```
- Layered depth without borders
- Frosted glass effect
- Works best on dark backgrounds
- Used by: jeton.com, Linear, Mercury

#### 3. Gradient Mesh Backgrounds
```css
.mesh-gradient {
  background:
    radial-gradient(at 40% 20%, #635bff 0%, transparent 50%),
    radial-gradient(at 80% 0%, #00d4ff 0%, transparent 50%),
    radial-gradient(at 0% 50%, #e85d4c 0%, transparent 50%);
}
```
- Multiple radial gradients
- Blur filters (60-100px)
- Subtle animation (scale/opacity)
- Used by: Stripe, Mercury, Vercel

#### 4. Scroll-Triggered Animations
```javascript
// GSAP ScrollTrigger pattern
gsap.fromTo(element,
  { opacity: 0, y: 80, rotationX: 15 },
  {
    opacity: 1, y: 0, rotationX: 0,
    duration: 1.2,
    ease: 'power3.out',
    scrollTrigger: {
      trigger: element,
      start: 'top 80%',
      scrub: 0.6
    }
  }
);
```
- 3D transforms for depth
- Scrub animations on desktop
- Toggle animations on mobile
- Used by: jeton.com, Apple, Linear

#### 5. Dark Mode as Default
```css
:root {
  --bg-primary: #08090a;
  --bg-secondary: #111113;
  --text-primary: #f5f5f5;
  --text-secondary: #a0a0a0;
  --accent: #635bff;
}
```
- Not pure black (#000) - use #08090a or similar
- High contrast text
- Colorful accents pop more
- Used by: Linear, Vercel, Ramp

   - New York + SF Pro

## Animation Patterns

### Premium Easing Functions
```css
/* jeton.com signature ease */
--ease-premium: cubic-bezier(0.23, 1, 0.32, 1);

/* Apple smooth */
--ease-apple: cubic-bezier(0.25, 0.1, 0.25, 1);

/* Stripe bounce */
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);

/* Linear subtle */
--ease-smooth: cubic-bezier(0.4, 0, 0.2, 1);
```

### 3D Entrance Animation
```javascript
// Premium entrance - used by ActionStack
gsap.fromTo(element, {
  opacity: 0,
  y: 80,
  scale: 0.9,
  rotationX: 15,
  rotationY: direction * 8,
  transformPerspective: 1200,
  transformOrigin: 'center bottom'
}, {
  opacity: 1,
  y: 0,
  scale: 1,
  rotationX: 0,
  rotationY: 0,
  duration: 1.4,
  ease: 'power3.out'
});
```

### Floating Card Animation
```javascript
// Subtle float - jeton.com style
gsap.to(card, {
  y: -8,
  duration: 3,
  repeat: -1,
  yoyo: true,
  ease: 'sine.inOut'
});
```

### Scroll Progress Animation
```javascript
// Scroll-linked reveal
gsap.to(element, {
  y: -60,
  scrollTrigger: {
    trigger: container,
    start: 'top top',
    end: 'bottom top',
    scrub: true
  }
});
```

## Key Takeaways for maemo

### What Works for Premium Consultancy

1. **Color Strategy**
   - Keep current coral (#e85d4c) as accent - it's distinctive
   - Consider darker ink (#0f172a → #000000) for more contrast
   - Add subtle gradient accents (Stripe-style) sparingly

2. **Typography Upgrades**
   - Current Outfit font is good, keep it
   - Increase headline sizes on desktop
